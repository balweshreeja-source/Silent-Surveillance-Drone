# Silent Surveillance Drone

## 📌 Project Overview
Silent Surveillance Drone is a hardware + software based drone designed for silent monitoring and surveillance in sensitive areas.

## 🎯 Features
- Silent Monitoring
- Motion Detection
- Alert System
- Temperature & Humidity Detection
- Remote Monitoring

## 🛠 Hardware Used
- Arduino Uno
- DHT11 Sensor
- IR Sensor
- Touch Sensor
- Buzzer
- Motors
- Wheels
- L298N Motor Driver
- Battery

## 💻 Software Used
- Arduino IDE
- Embedded C

## 👥 Team Name
GenZtrix

## 🚀 Purpose
To build a smart silent surveillance system for security and monitoring applications.
