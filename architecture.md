# Architecture Design

## System Workflow

Sensor Input → Arduino Processing → Motor Control → Alert System

### Components

1. Sensors
- IR Sensor
- DHT11
- Touch Sensor

2. Processing Unit
- Arduino Uno

3. Output
- Motors
- Buzzer
- LED

## Working

1. Sensors detect environment
2. Arduino processes data
3. Drone moves silently
4. Alert triggered if suspicious activity detected
